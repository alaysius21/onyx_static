# Onyx Paws Static HTML Version

This package is a static HTML/CSS/Bootstrap conversion of the uploaded app.

Pages:
- home.html
- service-detail.html
- booking-flow.html
- client-forms.html
- client-forms-success.html
- admin-login.html
- admin-dashboard.html

Demo login:
- Email: admin@onyxpaws.local
- Password: admin123

Notes:
- Bookings, client forms, and admin CRUD actions are stored in localStorage.
- This preserves the app interface and front-end flow without Flask/MySQL.
- PDF generation and real backend auth are simulated in the browser.
